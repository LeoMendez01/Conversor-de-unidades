# Conversor SV Pro

Aplicación estática, móvil y rápida para convertir unidades locales e internacionales.

## Modos

- **Unidades salvadoreñas**
- **Internacionales**
- **Todas**

## Regla de conversión

`value_in_base = value * from.to_base; result = value_in_base / to.to_base`

## Uso

1. Abre `index.html`.
2. Elige modo, categoría y unidades.
3. Ingresa valor y pulsa **Convertir**.
