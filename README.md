# Conversor de unidades

Aplicación web estática para convertir unidades usando el catálogo proporcionado.

## Uso

1. Abre `index.html` en tu navegador.
2. Selecciona una categoría.
3. Ingresa valor, unidad origen y unidad destino.

La regla aplicada es:

`value_in_base = value * from.to_base; result = value_in_base / to.to_base`
